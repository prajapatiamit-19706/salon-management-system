💇‍♂️ Salon Management System

A full-stack AI-powered Salon Management System built using the MERN stack. This application allows users to book salon services online, while providing admins with a powerful dashboard to manage appointments, staff, services, and payments.

---

🚀 Features

👤 User Features

- Browse salon services with pricing and duration
- Select preferred staff members
- Book appointments with date & time slots
- Secure payment integration (Razorpay - test mode)
- View booking history and download invoices
- Submit ratings and reviews after service completion
- Receive notifications via email/WhatsApp

---

🛠️ Admin Features

- Admin dashboard with analytics
- Manage services (add, update, delete)
- Manage staff and availability
- View and control all appointments
- Handle cancellations and completed bookings
- Track payments and billing

---

💳 Payment System

- Integrated Razorpay (test mode)
- Secure order creation and payment verification
- Automatic booking confirmation after payment

---

⭐ Feedback System

- Users can rate services and staff
- Reviews stored and used for analytics
- Helps identify top-rated services

---

📄 Legal Pages

- Privacy Policy
- Terms & Conditions
- Cancellation & Refund Policy

---

🧱 Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Payments: Razorpay
- Automation & AI: n8n, OpenAI (optional)
- Media Storage: Cloudinary (optional)

---

⚙️ Installation

# Clone repository
git clone https://github.com/your-username/salon-management-system.git

# Install backend
cd salon-backend
npm install

# Install frontend
cd ../salon-frontend
npm install

# Run backend
cd ../salon-backend
npm run dev

# Run frontend
cd ../salon-frontend
npm run dev

---

🔐 Environment Variables

Create ".env" files in both frontend and backend:

MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
